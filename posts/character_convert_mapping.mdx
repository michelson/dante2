# Continuous Blocks

This option provides some mapping on character combination on the beggining of pharagraph. For example you may want to create some ordered list just by typing "1." The following are some default character mappings:

## Defaults are :

```js
  character_convert_mapping: {
    '> ': "blockquote",
    '*.': "unordered-list-item",
    '* ': "unordered-list-item",
    '- ': "unordered-list-item",
    '1.': "ordered-list-item",
    '# ': 'header-one',
    '##': 'header-two',
    '==': "unstyled",
    '` ': "code-block"
  }
```

## Implement defaults

```js live=true
  <Dante 
    character_convert_mapping={ 
      {
        '* ': "blockquote",
        '##': 'header-two'
      }
   }
  />
```
